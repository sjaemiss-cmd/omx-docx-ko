Agents Reference

Oh My Codex organizes 30+ specialized AI personas into functional lanes. Use these agents to perform targeted tasks with maximum expertise.

How to Invoke

Load any agent persona using the /prompts:name command within the Codex CLI.

/prompts:architect "analyze the current architecture"
/prompts:executor "implement the login form"

#Build & Analysis

The core execution lane for planning, designing, and implementing software.

Implementation Core
AgentModelRole & When to UseExample
architect
 opus
Design system boundaries and draft high-level plans.
Use when: Starting a new feature or refactoring major components.
 /prompts:architect "review service boundaries"
planner
 opus
Sequence complex tasks into actionable steps.
Use when: You have a goal but need a concrete execution roadmap.
 /prompts:planner "plan migration"
executor

Write code and implement features following established patterns.
Use when: You are ready to write the actual implementation.
 /prompts:executor "add validation"

 /prompts:debugger "investigate flaky test"

 /prompts:verifier "verify release readiness"

#Review Lane

Quality gates focused on ensuring high standards, security, and performance.

Quality Gates
AgentModelRole & When to UseExample
code-reviewer

Use when: Preparing a Pull Request or reviewing a teammates change.
 /prompts:code-reviewer "comprehensive review"
security-reviewer
 sonnet
Scan for vulnerabilities, input validation, and auth gaps.
Use when: Shipping sensitive features or public APIs.
 /prompts:security-reviewer "security audit"
performance-reviewer
 sonnet
Identify bottlenecks, memory leaks, and optimizations.

 /prompts:performance-reviewer "profile hotspots"
